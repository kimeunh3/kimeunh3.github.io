## 코드  
```python
N = int(input())
stairs = [int(input()) for _ in range(N)]
cache = [[0 for _ in range(2)] for _ in range(N)]
cache[0][0] = stairs[0]
if N > 1:
    cache[1][0] = stairs[1]
    cache[1][1] = cache[0][0] + stairs[1]
    for i in range(2,N):
        cache[i][0] = max(cache[i-2]) + stairs[i]
        cache[i][1] = cache[i-1][0] + stairs[i]
print(max(cache[N-1]))
```
