## 문제
![BOJ_1436](https://user-images.githubusercontent.com/59808674/114321083-ec101580-9b53-11eb-8086-18b371339856.PNG)

## 풀이
```python
N = int(input())
cnt = 0
start = 666
while(1):
    start_str = str(start)
    for i in range(len(start_str)-2):
        if start_str[i:i+3] == "666":
            cnt += 1
            break
    if cnt == N:
        break
    start += 1

print(start)
```
