## 코드  
```python
N = int(input())
A = [tuple(map(int, input().split())) for _ in range(N)]
A.sort(key= lambda a: a[0])
cache = [1 for _ in range(N)]
for i in range(N):
    for j in range(i):
        if A[i][1] > A[j][1] and cache[i] <= cache[j]:
            cache[i] = cache[j] + 1
print(N-max(cache))
```
