__풀이__  
```python
N = int(input())
info = [ tuple(map(int, input().split())) for _ in range(N) ]
rank = [ 1 for _ in range(N) ]

for i in range(N):
    for j in range(N):
        if info[i][0] < info[j][0] and info[i][1] < info[j][1]:
            rank[i] += 1

print(*rank, sep=" ")
```
