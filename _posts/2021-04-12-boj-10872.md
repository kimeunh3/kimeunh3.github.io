## 풀이

팩토리얼의 공식은 아래와 같습니다.
 - N! = 1 * 2 * ... * (N-1) * N (0! = 1)  

이 공식은 아래 같이도 표현 가능합니다.
 - N! = (N-1)! * N  

그러므로 기저 사례(base case)는 N이 0일때 1을 반환하고,  
N과 N-1의 재귀호출을 곱한 값을 반환해줍니다.

## 코드
```python
def factorial(n):
    if n == 0: # 0! = 1
        return 1
    return n * factorial(n-1)

N = int(input())
print(factorial(N))
```
