## 코드  
```python
def pado(n):
    if cache[n] == -1:
        cache[n] = pado(n-2) + pado(n-3)
    return cache[n]
T = int(input())
for _ in range(T):
    N = int(input())
    cache = [-1 for _ in range(101)]
    cache[0] = 0
    cache[1] = 1
    cache[2] = 1
    print(pado(N))
```
