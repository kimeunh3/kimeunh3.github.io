## 코드  
```python
def fib(n):
    if cache[n] == -1:
        cache[n] = fib(n-1) + fib(n-2)
    return cache[n]
T = int(input())
for _ in range(T):
    cache = [-1] * 42
    cache[0] = 0
    cache[1] = 1
    N = int(input())
    if N == 0:
        print(1, 0)
    else:
        print(fib(N-1), fib(N))

```
