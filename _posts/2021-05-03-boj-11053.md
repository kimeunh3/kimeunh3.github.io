## 코드  
```python
N = int(input())
A = list(map(int, input().split()))
cache = [1 for _ in range(N)]
for i in range(N):
    for j in range(i):
        if A[i] > A[j] and cache[i] <= cache[j]:
            cache[i] = cache[j] + 1
print(max(cache))
```
