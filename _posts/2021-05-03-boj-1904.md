## 코드  
```python
N = int(input())
cache = [-1 for _ in range(N+2)]
cache[0] = 0
cache[1] = 1
for i in range(2,N+2):
    cache[i] = (cache[i-2] + cache[i-1]) % 15746
print(cache[N+1])
```
