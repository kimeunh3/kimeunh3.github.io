__풀이__  
```python
def decompose(n):
    if n // 10 == 0:
        return n % 10
    return n % 10 + decompose(n // 10)

N = int(input())
ans = 0
for i in range(N):
    if i + decompose(i) == N:
        ans = i
        break
print(ans)
```
